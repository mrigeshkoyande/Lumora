# Lumora Frontend

This is the frontend application for the Lumora platform, designed to interact with the Lumora backend services.

## Overview
The frontend connects to the backend API to provide a seamless user interface for:
1. **Health Surveillance Dashboard**: Visualizing health risk analysis, weather data, AQI, and news snapshots.
2. **Inventory & Procurement Orchestration**: Monitoring real-time stock levels across health zones and viewing AI-generated procurement alerts and insights generated by the Agno (Phidata) agent.

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation & Run

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start Dev Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

### Environment Variables
Copy `.env.example` to `.env` and fill in the required variables (e.g., `VITE_API_URL` to point to the backend if running in production, defaults to proxying `/api` -> `http://127.0.0.1:8000` in dev).
